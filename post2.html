How GitHub Pages Hosts Websites

Posted by Alex on February 4, 2025

GitHub Pages is a free static web hosting service that serves websites directly from a GitHub repository.

How It Works

  • You push HTML, CSS, and JavaScript files to a GitHub repository.
  • GitHub Pages generates a website using your repository.
  • The site is accessible via a "github.io" subdomain.

Networking Behind GitHub Pages

When a user visits a GitHub Pages site:

  • The browser requests the page using DNS resolution to find the server's IP address.
  • GitHub's network delivers the content efficiently.
  • HTTPS encrypts the connection for security.

This is why I chose GitHub Pages and it's simple to use for making static websites.
